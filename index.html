Hardware + Software

Cub & CubOS

Coupled hardware and software that turn a CNC gantry into a self-driving lab.

Cub

Modified hardware platform

A modified CNC gantry with custom instrument mounts, labware, and safety enclosures — purpose-built for autonomous characterization in materials science and chemistry labs.

CubOS

Open-source Python package

Define entire experiment workflows in simple YAML configs. CubOS handles motion control, instrument coordination, data collection, and closed-loop optimization — no low-level code required.
